How We Use the Pinterest API

Our application uses the Pinterest API to:

  • Retrieve your public Pinterest boards and pins only after you explicitly grant OAuth permission
  • Help you save and organize wall art inspiration to your Pinterest boards
  • Display relevant decor inspiration based on your saved pins to match our art print collection

All API access follows Pinterest's strict Developer Terms. You can revoke our access at any time from your Pinterest account security settings.
